# Simple-snake-game
My first tinkering with windows curses module.

////////////////////////////////////////////////////////////////////////////////////////////////////////
Do Note :-
In order to install curses module using pip:-

For Windows based systems 
```prompt
pip install windows-curses

```


For Unix/Linux based systems 
```terminal
pip install curses

```
